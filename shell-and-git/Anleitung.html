Anleitung Repository

Lokales Repository erstellen

  • gehe im Terminal zum Ordner der zum Repository werden soll
  • man sollte nie ein Repository innerhalb eines bestehenden Repository erstellen
  • mit git init wird der Ordner als Repository definiert
  • folgende Nachricht sollte erscheinen: 
    Initialized empty Git repository in /Users/your-username/path_to_folder/session-notebook/.git/

Remote Repository anlegen

  • in Github neues Repository anlegen
  • es sollte ein öffentliches Repository sein und den selben Namen wie der lokale Ordner besitzen

Repository verbinden

  • SSH Adresse aus gerade angelegtem remote Repository kopieren
  • diese als remtoe repository "origin" zum lokalen Repository hinzufügen über git remote add origin SSH Adresse
  • mit git remote -v überprüfen, ob das Repository erfolgreich verbunden wurde (eine "fetch" und eine "push" Adresse sollten im Terminal angezeigt werden)
  • git add . um alle Änderungen zu stagen
  • git commit -m "Beschreibung" um alles zu übergeben
  • git push um alles hochzuladen - beim ersten Mal muss man genau spezifizieren wohin man hochladen will mit git push -u origin main
